Please use the following command to run the program on the command line

There is an inputTest.txt file which is read by default but you can pass the 
file name separately as an argument to "npm run start".

Note: Assuming you are going to run this program on Windows,
please install NodeJS on your machine if its not installed already

// This is to install packages although I haven't used any except for testing
1) npm i

// This command reads inputTest.txt by default
2) npm run start

// This command takes an input text to be read by the program
3) npm run start -- input.txt

// This is for running test cases
4) npm run test

// This command is for running test coverage
5) npm run test:coverage

Please let me know If you are facing any issues while running this code.
